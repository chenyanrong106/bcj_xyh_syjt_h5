活动规则

活动时间：

2018年5月8日- 2018年5月28日

奖品设置：
  • 一等奖1名：平安司庆金条10g
  • 二等奖10名：平安司庆定制香水1瓶
  • 三等奖100名：平安司庆银条10g
  • 参与奖： 上海家化商城优惠券（满999元-100元 / 满499元-50元 / 满99元-10元）
活动机制：
  • ~参与活动，摇动手机递增点亮平安大楼，直到118层全部点亮，即可抽奖，每人每天可抽奖10次；
  • ~每位玩家在进入活动时，共有10秒倒计时，摇动频率越高，点亮楼层越多；
  • ~每转发至朋友圈或朋友即可获得5秒延时，每次参与活动最多可有15秒。在15秒内未能全部点亮，再次进入活动时，将会重新开始点亮楼层。 注：若中途退出活动，楼层点亮数字将清零。
奖品发放：

获得实物奖品，我们将在活动结束后1个月内寄送，届时请注意查收。请中奖玩家在中奖后跳转页面内及时领取并填写完整个人信息，领取时间截止5月28日活动结束，过期不领、因信息填写错误或不完整所造成的奖品发放失败，视为自动弃奖不予补寄。奖品一经寄出，无质量问题不予退换。
